# ecomm-python-sql
This project analyzes e-commerce product and cart data to understand how product characteristics influence purchasing behavior and cart value. Using data from the DummyJSON API, product-level attributes (such as price, rating, discount percentage, and inventory) were combined with cart-level transaction data to create a unified dataset where each row represents a product appearing within a customer cart. The analysis uses both Python (pandas, seaborn, scikit-learn) and SQL (SQLite) to perform data cleaning, feature engineering, exploratory data analysis, and regression modeling. SQL queries are used to analyze relationships between products and cart behavior through joins and aggregations, while Python visualizations and models explore how features such as price buckets, discounts, and quantity interactions relate to cart value. The goal of the project is to provide insights relevant to e-commerce merchandising and growth analysts, highlighting which product characteristics and categories are associated with higher cart totals and stronger purchasing behavior.
